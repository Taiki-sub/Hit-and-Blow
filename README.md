# Hit-and-Blow

## 🔢ゲーム概要
プレイヤー同士でお互いの3桁の数字を当てるゲームです。

予めお互いに3桁の数字を決めてそれぞれ相手の数字が何なのか順番に推理して先に相手の数字を当てた方が勝利となります。

※同じタイミングで当てた場合はドローとなります。
